#### Node Modules

##### 3 kinds of Node

- file-based Module
- core Modules
  - part of core Node
  - path, fs, os, util...
- external Node modules
  - 3th party modules,
  - installed using NPM
  - node_modules folder in your Node application

##### Using Node Modules

By using require function

File-based: require('./module_name')

core/External: require('module_name')
